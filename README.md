# AI Outage Bot

AI-powered livestream bot that monitors global tech outages, summarizes them, and streams live updates via OBS.

## Current Build

- ✅ Downdetector scraping (Playwright)
- ✅ OpenAI GPT-4o summarization
- ✅ OBS integration (WebSocket)
- ✅ Live overlay updates
- ✅ Amazon Polly TTS
